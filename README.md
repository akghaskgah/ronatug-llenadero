# Ronatug

Este repositorio ahora está enfocado en la aplicación web estática que se sirve desde la carpeta `www/`.

## Ejecutar la app

```powershell
npm install
npm run serve
```

Después de eso, abre `http://localhost:8081` en tu navegador.

## Qué se eliminó

- Se eliminó el backend antiguo de `backend/`.
- Se limpió el `package.json` para mantener solo la app web estática.

## Estructura principal

- `www/index.html`
- `www/app.js`
- `www/styles.css`
- `www/firebase.js`

Esta es la app que debe verse con `npm run serve`.
